    - Data Types:
        1. Primitive Data Types
            1. Number: 0 - 9; 653, -356, 465.433, 46e643

            1. Array

-> Identifiers
    1. Every Identifier should start with either a - z, A - Z, _
    2. From 2nd Character onwards., it can be a - z, A - Z, 0 - 9, _
    3. No special symbols except _
    4. No spaces
    5. Combination of lowercase & uppercase of alphabets

    eg., fullname, FULLNAME, fullName, full_name, num1, n1, _num,
